## Instalar repositorios AUR

[Instalación yay](https://itsfoss.com/install-yay-arch-linux/)

Instalar FreeTube:
```sh
yay -S freetube
```
Instalar Zen-browser:
```sh
yay -S zen-browser-bin
```
